== Description ==

在用户留言的时候，通过邮箱地址匹配QQ或新浪微博头像. 此插件主要针对的是中文博客. 详情请查看: http://blog.i7avatar.com/what-is-i7avatar.html.

== Installation ==

1. 下载i7avatar.zip.
1. 从插件管理界面上传插件.
1. 点击`现在安装`.
1. 在后台管理面板里的`设置` -> `i7avatar`可以找到插件的设置选项.

或

1. 解压缩下载的i7avatar.zip.
1. 将解压缩后得到的文件夹上传到WordPress `/wp-content/plugins/` 目录下.
1. 在WordPress的后台`管理插件`页面启用`i7avatar`.
